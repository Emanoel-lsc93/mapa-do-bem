# 04 — Backlog e MVP

## MVP (mínimo que presta)
1. Página inicial com categorias (Saúde, Assistência Social, Educação, Trabalho, Documentos)
2. Busca por palavra-chave
3. Lista de serviços em cards com informações úteis
4. Link de fonte oficial em cada card
5. Repositório organizado com documentação

## Itens para versão 2
- Filtro por categoria/bairro
- Links de rota (Google Maps)
- Acessibilidade (contraste, tamanho de fonte, navegação)
- Expansão da base de serviços com curadoria
