# 05 — Relatório do Processo (Projeto Integrador)

## 1) Tema social e projeto
O projeto aborda a desinformação e a dificuldade de acesso a serviços e benefícios locais. Foi definido o **Mapa do Bem**, uma plataforma para centralizar informações essenciais com linguagem acessível e uso de fontes confiáveis.

## 2) Objetivos e público
O objetivo é facilitar o acesso a serviços e direitos organizando informações por categoria e permitindo busca rápida. O público beneficiado são moradores da cidade/região, especialmente famílias, gestantes, jovens e pessoas que precisam de orientação prática.

## 3) Desenvolvimento técnico (fase inicial)
Foi estruturado um repositório no GitHub com documentação inicial, planejamento do MVP e definição das tecnologias a serem utilizadas. A proposta prioriza uma primeira versão simples (MVP) para entrega rápida e evolução incremental.

## 4) Organização no GitHub
O repositório foi organizado com:
- README com visão geral, objetivos, público e roadmap
- pasta /docs com documentação do projeto e do processo

## 5) Próximos passos
- Implementar o MVP (busca + cards + base inicial)
- Expandir e validar dados com fontes oficiais
- Melhorar usabilidade e acessibilidade
- Publicar atualização no LinkedIn para divulgar e coletar feedback
