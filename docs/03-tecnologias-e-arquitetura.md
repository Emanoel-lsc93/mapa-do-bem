# 03 — Tecnologias e Arquitetura (Planejamento)

## Estratégia técnica (fase inicial)
Construir um MVP simples com dados em **JSON/CSV**, permitindo iniciar rápido e entregar valor.

## Tecnologias
- Front-end: React (opcional) ou HTML/CSS/JS
- Dados: JSON/CSV
- Controle de versão: Git + GitHub
- Deploy: GitHub Pages (front) / Netlify

## Estrutura de dados (exemplo)
Cada serviço deve conter:
- nome
- categoria
- endereço
- horário
- documentos necessários
- contato (se houver)
- fonte oficial (link)

## Boas práticas
- README claro
- documentação em /docs
- commits com mensagens objetivas
- organização por pastas
