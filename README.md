# Mapa do Bem (Projeto Integrador)

## Visão geral
O **Mapa do Bem** é um projeto com impacto social que centraliza informações essenciais (serviços públicos, direitos e orientações) de forma simples, acessível e com fontes confiáveis.

A proposta é reduzir a desinformação e o tempo perdido procurando informações espalhadas, facilitando o acesso a serviços e benefícios.

## Problema social
Muita gente deixa de acessar serviços e direitos por falta de informação clara, atualizada e fácil de encontrar.

## Objetivo
Criar uma plataforma/portal que organize serviços por categoria (saúde, assistência social, educação, trabalho, documentação) e permita busca rápida por palavra-chave.

## Público beneficiado
Moradores da cidade/região, especialmente famílias, gestantes, jovens e pessoas que precisam de orientação prática.

## Tecnologias (planejadas)
- Front-end: React ou HTML/CSS/JS
- Dados: JSON/CSV (fase inicial)
- Versionamento: GitHub
- Publicação: GitHub Pages/Netlify (front)

## Entregas (MVP)
- Categorias principais
- Busca por palavra-chave
- Cards com informações: nome, endereço, horário, documentos necessários e fonte oficial
- Documentação do projeto (pasta /docs)

## Roadmap
- v1 (MVP): base inicial + busca + cards
- v2: filtros por bairro/categoria + mapa (links de rota) + melhoria de acessibilidade
- v3: atualização periódica e colaboração da comunidade (com curadoria)
